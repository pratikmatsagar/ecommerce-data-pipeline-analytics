# E-Commerce Data Pipeline & Analytics Platform

Production-style ETL and analytics workflow project built using PySpark, Delta Lake concepts, SQL, and Power BI following Bronze-Silver-Gold Medallion Architecture.

---

## Architecture

Bronze Layer → Raw Data Ingestion  
Silver Layer → Data Cleaning & Transformation  
Gold Layer → Analytics-ready Reporting Datasets

---

## Tech Stack

- PySpark
- Delta Lake
- SQL
- Power BI
- Git & GitHub
- Azure Data Engineering Concepts

---

## Features

- Incremental ETL workflows
- Bronze/Silver/Gold architecture
- Structured data transformation
- KPI reporting workflows
- Analytics-ready datasets
- Business intelligence dashboards

---

## Project Workflow

1. Raw e-commerce datasets ingested into Bronze layer  
2. Data cleaning and transformation performed in Silver layer  
3. Gold layer datasets prepared for analytics and dashboard reporting  
4. Power BI dashboards built for KPI and sales analysis
